HTML 常学内容

基础标签

  • 标题标签：h1 到 h6
  • 段落标签：p
  • 强调标签：strong、em
  • 链接标签：a
  • 图片标签：img

学习步骤

  1. 先理解 HTML 文档的整体结构
  2. 再学习常用文本标签和列表标签
  3. 然后掌握表格、表单和语义化标签
  4. 最后通过项目练习巩固

学习计划表

阶段 学习内容 目标
第一天 HTML 基本结构 看懂完整网页骨架
第二天 常用标签 会写标题、段落、列表、链接、图片
第三天 表格与表单 能制作信息展示和输入页面
